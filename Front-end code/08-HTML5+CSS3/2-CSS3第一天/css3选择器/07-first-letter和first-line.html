张栋梁 1981年11月29日 马来西亚沙捞越州古晋市 马来西亚 张栋梁，1981年11月29日出生于马来西亚沙捞越州古晋市，祖籍福建，马来西亚歌手、演员。 2002年，参加Astro新秀大赛，演唱《黄昏》等歌曲并获得冠军，从而正式出道。 2003年，发行首张个人EP《张栋梁EP》。2004年，凭借首张个人音乐专辑《首选》获得第12届新加坡金曲奖“最佳新人奖” ； 同年，担任马来西亚八度空间电视台音乐类节目《八号风球》的主持人。2006年，主演浪漫青春偶像剧《微笑PASTA》。 2008年，主演爱情偶像剧《无敌珊宝妹》 。2010年，主演都市爱情偶像剧《女王不下班》。2011年，加盟种子音乐并推出专辑《别再惊动爱情》， 凭借专辑同名主打歌《别再惊动爱情》获得音乐先锋榜“年度港台十大先锋金曲。2013年，主演音乐偶像剧《K歌·情人·梦》。 2015年8月，推出EP《长假》；9月，主演偶像剧《爱情有没友-星座爱情水瓶女》。 2016年9月14日，凭借专辑《怎样的张栋梁》获得马来西亚2016AIM中文音乐颁奖典礼最佳男歌手、最佳专辑等奖。2018年1月27日， 在台北举行《口袋的五度世界》演唱会。
